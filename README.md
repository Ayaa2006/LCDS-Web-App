# 📸 LCDS - LaCasaDeSelfie (Stage de Fin d'Études)

## 📝 Présentation du Projet
Projet réalisé dans le cadre de mon stage chez **LCDS**. [cite_start]Il s'agit d'une application web et mobile complète intégrant des fonctionnalités avancées pour l'expérience utilisateur[cite: 11, 13].

## ✨ Fonctionnalités Clés
* [cite_start]**Réservation Intelligente :** Système de gestion des réservations optimisé.
* [cite_start]**Paiements Sécurisés :** Intégration de la passerelle de paiement **PayPal**.
* [cite_start]**Expérience Immersive :** Vue à **360°** intégrée avec **Parolens.js**.
* [cite_start]**Innovation IA :** Module d'**Intelligence Artificielle** pour l'édition et l'amélioration de photos.
* [cite_start]**Gamification :** Mise en place de mécanismes de jeu pour fidéliser les utilisateurs.

## 🛠️ Technologies Utilisées
* [cite_start]**Backend :** Laravel (PHP)[cite: 13, 27].
* [cite_start]**Frontend :** React.js[cite: 13, 27].
* [cite_start]**Base de données :** MySQL[cite: 13, 28].
* [cite_start]**IA & 3D :** Parolens.js et modules d'édition photo IA.
